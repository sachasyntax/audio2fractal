Audio–Visual Morphologies  
=========================

Author: Sacha Syntax  
Year: 2025  
License: MIT  

----------------------------------------
DESCRIPTION
----------------------------------------

This repository contains a collection of Python scripts that generate visual structures directly from the raw binary data of audio files.

Each script interprets sound not through frequency or rhythm, but as a flow of bytes — converting digital audio into unpredictable, emergent visual forms.

Depending on the script, the resulting images may take the shape of fractal patterns, neural-like networks, or dense annotated data clusters.

----------------------------------------
INCLUDED SCRIPTS
----------------------------------------

audio2fractal.py  
    Generates a fractal-inspired visual map of the audio data.  
    The resulting image blends recursive transformations and organic symmetry.

audio2neural.py  
    Translates the raw byte structure of the sound into a 3D neural-style graph.  
    Nodes and connections emerge based on the data’s internal randomness.

audio2neural2.py  
    A dense, annotated version of the neural network visualization.  
    Each node is labeled with its numeric data, producing a visually overloaded information map.

----------------------------------------
REQUIREMENTS
----------------------------------------

1. Python 3  
   Make sure you have Python 3 installed.  
   Check the version in your terminal:
       python3 --version  
   It should be version 3.9 or higher.

2. Python Libraries  
   Install the required dependencies using pip:
       python3 -m pip install numpy matplotlib pillow

   Optional (for video generation or animation features):
       python3 -m pip install opencv-python

----------------------------------------
HOW TO USE
----------------------------------------

1. Open the Terminal and move into the folder containing the scripts:
       cd ~/Desktop/audio-visual

2. Run one of the scripts with an audio file and specify the output image:
       python3 audio2fractal.py input.wav output.jpeg

   Example:
       python3 audio2neural2.py samples/audio.wav outputs/output.jpeg

   The script will:
   - Read the audio file’s bytes  
   - Transform them into coordinate, color, and structural data  
   - Render an abstract visual interpretation  
   - Save the result as a high-resolution image

----------------------------------------
OUTPUT FORMATS
----------------------------------------

By default, images are saved as JPEG files.  
You can also modify the script to save as PNG, SVG, or PDF:

    plt.savefig("output.png", dpi=300)
    plt.savefig("output.svg")
    plt.savefig("output.pdf")

----------------------------------------
CONCEPT
----------------------------------------

These visualizations are not spectrograms or waveform analyses.  
They are abstract morphologies — graphic systems generated by reinterpreting raw data as spatial and chromatic phenomena.

Each execution creates a different image, even with the same audio source, due to the random and recursive mapping used to interpret the byte patterns.
